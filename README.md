# Portfolio Website

My personal portfolio site. Built with plain HTML, CSS, and JavaScript — no frameworks, no nonsense.

**Live at:** [showmikbhowmik.github.io/portfolio](https://showmikbhowmik.github.io/portfolio/)

## What's in here

- **index.html** — main page structure
- **styles.css** — all the styling and animations
- **script.js** — interactive bits (scroll animations, particles, etc.)

## Sections

- Hero / intro
- Featured projects (VisionVoice, CodeChef)
- Work experience
- Skills
- Contact links

## Running locally

Just open `index.html` in a browser. That's it.

## Tech

- HTML / CSS / JS
- Google Fonts (Poppins)
